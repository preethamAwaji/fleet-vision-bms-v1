# Dashboard Screenshots

## Instructions to Add Screenshots

Visit the live dashboard at: **https://fleet-vision-bms.netlify.app**

Take screenshots of the following pages and save them in this folder:

### 1. dashboard_main.png
- Main fleet overview page
- Shows all vehicles with battery status
- Real-time updates visible

### 2. vehicle_detail.png
- Individual vehicle detail page
- Click on any vehicle (e.g., EV-001)
- Shows voltage, temperature, current charts
- GPS location map

### 3. telemetry.png
- Telemetry/Historical data page
- Shows time-series charts
- Battery performance over time

### 4. analytics.png
- Fleet analytics page
- Shows fleet-wide statistics
- Comparative analysis

## How to Take Screenshots

### Method 1: Windows Snipping Tool
1. Press `Windows + Shift + S`
2. Select area to capture
3. Save as PNG in this folder

### Method 2: Browser Developer Tools
1. Open dashboard in browser
2. Press `F12` (Developer Tools)
3. Click device toolbar icon (mobile view)
4. Set resolution to 1920x1080
5. Take screenshot
6. Save in this folder

### Method 3: Full Page Screenshot
1. Open dashboard
2. Press `F12`
3. Press `Ctrl + Shift + P`
4. Type "screenshot"
5. Select "Capture full size screenshot"
6. Save in this folder
