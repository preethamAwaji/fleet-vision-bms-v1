# EV Fleet - Intelligent Electric Vehicle Fleet Management

## Project Overview

A comprehensive electric vehicle fleet management system built with modern web technologies. Monitor battery health, optimize charging, track vehicles in real-time, and keep your fleet running at peak efficiency.

## Features

- **Role-based Navigation**: Tailored interfaces for Fleet Operators, Maintenance Engineers, and Admins
- **Real-time Monitoring**: Live vehicle tracking and battery health monitoring
- **Intelligent Charging**: Optimize charging schedules and station utilization
- **Maintenance Management**: Proactive maintenance scheduling and tracking
- **Analytics & Reporting**: Comprehensive fleet performance insights

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **UI Framework**: Tailwind CSS with shadcn/ui components
- **Build Tool**: Vite
- **State Management**: TanStack Query + React Context
- **Routing**: React Router v6
- **Icons**: Lucide React
- **Charts**: Recharts

## Quick Start

To get started with this project:

```bash
# Step 1: Clone the repository
git clone https://github.com/Mallikarjun-P-23/Fleet_Intelligence_Platform.git

# Step 2: Navigate to the project directory
cd Fleet_Intelligence_Platform

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production  
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode

## Role-Based Access

The application supports three user roles:

1. **Fleet Operator**: Focus on fleet monitoring, analytics, and reporting
2. **Maintenance Engineer**: Specializes in alerts, diagnostics, and maintenance  
3. **Admin**: Complete system administration and configuration
